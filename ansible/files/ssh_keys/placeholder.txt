📁 Directorio para Claves SSH Públicas
==========================================

Este directorio es para almacenar las llaves SSH **públicas** de los usuarios autorizados.

⚠️  IMPORTANTE: 
- Solo subir llaves SSH **PÚBLICAS** (archivos .pub)
- NUNCA subir llaves privadas al repositorio
- Un archivo por usuario

📝 Instrucciones de uso:
1. Crea un archivo con el formato: nombre_usuario.pub
2. Pega el contenido de tu llave SSH pública dentro del archivo
3. El contenido debe comenzar con ssh-rsa, ssh-ed25519, etc.

💡 Ejemplo de contenido de archivo (usuario: juan.pub):
ssh-rsa AAAAB3NzaC1yc2EAAAADAQABAAABgQC7... juan@ejemplo.com

🔧 Para generar tu par de claves SSH (si no las tienes):
ssh-keygen -t ed25519 -C "tu-email@ejemplo.com"

📋 Para obtener tu clave pública:
cat ~/.ssh/id_ed25519.pub  # o id_rsa.pub si usas RSA

🎯 Ejemplos de archivos que deberías crear aquí:
- admin.pub
- developer.pub  
- devops.pub
- maria_garcia.pub
- etc.

Los roles de Ansible leerán automáticamente estos archivos y configurarán
el acceso SSH para los usuarios correspondientes.
